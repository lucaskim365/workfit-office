# MES Platform

스마트 MES 생산 플랫폼 — 풀스코프 제조실행시스템(8개 모듈 / 약 170개 화면).

MVP 프로토타입(React 19 + TypeScript + Vite + Firebase)의 검증된 기술 기반 위에,
풀스코프 와이어프레임의 화면 설계를 정본으로 하여 실제 제품으로 확장 개발합니다.

## 문서
- [프로젝트 개발 계획서](./MES_프로젝트_개발계획서.md) — 범위·아키텍처·로드맵·개발/배포 환경

## 기술 스택 (확정)
- **프론트엔드**: React 19 + TypeScript + Vite (SPA)
- **스타일**: Tailwind CSS 4 (와이어프레임 디자인 토큰 기반)
- **백엔드**: Firebase (Firestore / Auth / Storage) + Cloud Functions
- **배포**: Vercel (GitHub 연동, PR Preview / main Production)
