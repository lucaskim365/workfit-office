# 📋 선택 필드 존재 문서양식별 조건부 필드 추천 및 가이드

본 문서는 결재서식 중 **'선택(Choice)'** 타입의 입력필드가 존재하는 양식들을 대상으로, 기안자가 선택한 항목에 맞추어 유기적으로 화면에 노출하면 효율적인 **추천 조건부 필드 양식 기획안**을 정리한 문서입니다.

---

## 1. 휴가원 (Leave Request)
* **기준 선택 필드**: `leaveType (휴가종류)` -> `['연차', '반차', '병가', '경조사', '기타']`

| 선택 항목 | 추천 노출 필드 | 필드 타입 | 필수 여부 | 비고 / 설명 |
| :--- | :--- | :--- | :--- | :--- |
| **반차** | 오전/오후 구분 | 선택 | 필수 | 오전반차 / 오후반차 선택 드롭다운 |
| **경조사** | 경조사 구분 | 선택 | 필수 | 결혼, 수해/상례, 회갑 등 세부 종류 |
| | 대상자와의 관계 | 텍스트 | 필수 | 예: 본인, 부모, 배우자 등 |
| **병가** | 진단서 첨부 | 텍스트 | 필수 | 사내 드라이브 링크 또는 파일명 기재 |

---

## 2. 국내출장 신청서 (Domestic Business Trip Request)
* **기준 선택 필드**: `purpose (출장 목적)` -> `['영업', '기술지원', '교육', '회의', '기타']`

| 선택 항목 | 추천 노출 필드 | 필드 타입 | 필수 여부 | 비고 / 설명 |
| :--- | :--- | :--- | :--- | :--- |
| **영업** | 방문 고객사명 | 텍스트 | 필수 | 영업 대상 업체 및 담당자 정보 입력 |
| | 영업 기회 단계 | 선택 | 선택 | 제안, 견적, 계약 등 진행 현황 |
| **기술지원** | 대상 장비/시스템명 | 텍스트 | 필수 | 장애 대응 및 설치를 진행할 기기명 |
| | 이슈/장애 번호 | 텍스트 | 선택 | 사내 ITSM 시스템의 장애 티켓 ID |
| **교육** | 교육 과정명 | 텍스트 | 필수 | 수강 예정인 교육 프로그램 공식 명칭 |

---

## 3. 해외출장 신청서 (Overseas Business Trip Request)
* **기준 선택 필드**: `overseasType (해외출장 구분)` -> `['일반 해외출장', '비자 및 여권 발급']`

| 선택 항목 | 추천 노출 필드 | 필드 타입 | 필수 여부 | 비고 / 설명 |
| :--- | :--- | :--- | :--- | :--- |
| **비자 및 여권 발급** | 발급 대상 구분 | 선택 | 필수 | 신규 여권 / 여권 갱신 / 신규 비자 |
| | 비자 종류 | 텍스트 | 필수 | 예: B1/B2, F1, ESTA 등 |
| | 만료 예정일 | 날짜 | 필수 | 기존 여권/비자의 유효기간 |
| | 발급 예상 수수료 | 금액 | 필수 | 청구 예정 수수료 |

---

## 4. 식대 신청서 (Meal Expense Request)
* **기준 선택 필드**: `expenseType (식대 구분)` -> `['간식대', '외근식대', '야근식대']`

| 선택 항목 | 추천 노출 필드 | 필드 타입 | 필수 여부 | 비고 / 설명 |
| :--- | :--- | :--- | :--- | :--- |
| **야근식대** | 연장근무 시간(H) | 숫자 | 필수 | 신청 당일 연장근로 시간 (예: 3) |
| | 근무 확인서 | 텍스트 | 선택 | 사내 드라이브 링크 또는 증빙 파일명 기재 |
| **외근식대** | 외근 목적지 | 텍스트 | 필수 | 출장지가 아닌 일상적인 외근 장소 |

---

## 5. 회식비 청구서 (Dinner Expense Request)
* **기준 선택 필드**: `expenseType (회식 구분)` -> `['부서원 회식', '특별회식(전체 회식)']`

| 선택 항목 | 추천 노출 필드 | 필드 타입 | 필수 여부 | 비고 / 설명 |
| :--- | :--- | :--- | :--- | :--- |
| **부서원 회식** | 참석 인원 수 | 숫자 | 필수 | 비용 정산(인당 한도)용 실제 참석인원 |
| **특별회식(전체 회식)** | 공식 행사/포상명 | 텍스트 | 필수 | 송년회, 분기 목표 달성 포상 등 행사명 |
| | 대상 본부/팀 범위 | 텍스트 | 필수 | 참석한 조직들의 범위 명시 |

---

## 6. 회의비 청구서 (Meeting Expense Request)
* **기준 선택 필드**: `expenseType (회의 구분)` -> `['부서회의비', '고객미팅비']`

| 선택 항목 | 추천 노출 필드 | 필드 타입 | 필수 여부 | 비고 / 설명 |
| :--- | :--- | :--- | :--- | :--- |
| **고객미팅비** | 외부 참석자 소속/이름 | 텍스트 | 필수 | 미팅을 함께한 파트너사 및 바이어 정보 |
| | 미팅 장소 | 텍스트 | 필수 | 식음료를 결제한 장소 또는 카페명 |

| 선택 항목 | 추천 노출 필드 | 필드 타입 | 필수 여부 | 비고 / 설명 |

| | 발송 방식 | 선택 | 필수 | 등기우편 / 직접전달 / 이메일 / 팩스 |

## 8. 인장날인 신청서 (Corporate Seal Request)
* **기준 선택 필드**: `stampType (인장 종류)` -> `['법인인감', '사용인감', '대표이사직인']`

| 선택 항목 | 추천 노출 필드 | 필드 타입 | 필수 여부 | 비고 / 설명 |

| | 제출 목적 구분 | 선택 | 필수 | 입찰 참가 / 계약 체결 / 관공서 제출 |
