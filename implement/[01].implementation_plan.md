# [1단계] UI/UX 레이아웃 & 테마 개선 개발계획서

본 계획서는 `계획서.md`에 명시된 **1단계: UI/UX 레이아웃 & 테마 개선**의 상세 설계 및 기술적 구현 방안을 다룹니다. 사용자가 시스템의 변화를 즉각적으로 체감하고, 조작 편의성을 극대화하기 위한 구조 개선과 테마 커스터마이징을 수행합니다.

---

## 🛠️ 주요 변경 사항 및 기술 설계

### 1. MES ➔ Intranet 명칭 변경
시스템의 정체성을 사내망(Intranet)으로 전면 이관하기 위해 화면 내 모든 'MES' 및 'Smart Factory' 표기를 변경합니다.

- **대상 파일**:
  - `index.html`: `<title>`을 `WorkFit Intranet`으로 변경
  - [Topbar.tsx](file:///c:/WorkFit/전자결재시스템/workfit-office/src/app/shell/Topbar.tsx): 로고 영역의 `WorkFitMES` ➔ `WorkFit Intranet`, `Smart Factory Suite` ➔ `Enterprise Portal Suite` 변경
  - [LoginScreen.tsx](file:///c:/WorkFit/전자결재시스템/workfit-office/src/app/auth/LoginScreen.tsx): 기본 fallback 회사명/시스템명 변경 (`스마트 MES` ➔ `WorkFit`, `생산 관리 플랫폼` ➔ `Intranet Portal`)
  - [companyInfo.seed.ts](file:///c:/WorkFit/전자결재시스템/workfit-office/src/data/seeds/companyInfo.seed.ts): `sysName` 및 `reportHeader`, `docFooter`를 `WorkFit Intranet`으로 변경

---

### 2. 우측 퀵 도크(Quick Dock) ➔ 상단 헤더(Topbar) 이관
기존의 우측 세로 탭 책갈피를 완전히 제거하고, 상단 헤더(`Topbar.tsx`) 영역에 아이콘을 탑재하여 퀵 드로어(Quick Dock Drawer)를 토글 제어합니다.

- **기술 구현**:
  - `AppShell.tsx`에서 퀵 도크의 열림 상태(`dockOpen: 'gw' | 'bot' | 'msg' | null`)를 전역 상태로 관리합니다.
  - `Topbar.tsx`의 날짜 및 계정 영역 좌측에 그룹웨어, 위디, 메신저를 호출할 수 있는 세련된 3개의 아이콘 버튼을 추가합니다.
  - `QuickDock.tsx` 내의 vertical bookmark tabs(세로 책갈피 탭)을 완전히 삭제하고, 오직 slide panel과 backdrop(dim)만 유지합니다.

---

### 3. 창 닫기 UX 개선 (✕ 기호 통일)
- **기술 구현**:
  - `QuickDock.tsx` 내에 존재하는 슬라이드 패널 닫기 및 프로필 헤더 닫기 버튼인 `›` 문자를 직관적이고 미려한 `✕` 문자(U+2715)로 통일하여 일체감을 줍니다.

---

### 4. 테마 색상 커스터마이징 및 화사한 테마 개선 (localStorage 연동)
사용자가 직접 사내망의 스타일을 제어할 수 있도록 메인 색상(Main Theme) 및 포인트 색상(Point Color) 선택기를 구현합니다.

- **기본 테마 개선**:
  - `index.css`에서 다소 칙칙한 회색 톤(`--color-bg`, `--color-bg-deep`)을 보다 화사하고 깨끗한 톤으로 기본값을 변경합니다.
- **테마 커스터마이저 모달 개발**:
  - 사용자가 개인 선호에 맞게 색상을 변경할 수 있는 `ThemeCustomizerModal.tsx`를 신규 생성합니다.
  - 제공 테마 후보군:
    - **메인 테마**: 클래식 딥블루 (기본), 에메랄드 그린, 세련된 바이올렛, 포근한 모카
    - **포인트 색상**: 클래식 틸 (기본), 로얄 블루, 비비드 앰버, 로즈 레드
  - 설정값을 `localStorage`에 즉시 저장하고, 변경 시 `document.documentElement.style.setProperty`를 통해 전역 CSS custom properties를 실시간 업데이트합니다.
  - `UserMenu.tsx` 내에 "테마 설정" 항목을 추가하여 이 모달을 손쉽게 열 수 있도록 합니다.

## 🧪 Verification Plan

### Manual Verification
1. **로컬 개발 서버 실행**: `npm run dev` 실행 후 접속.
2. **명칭 변경 확인**: 로그인 화면 및 상단 탑바에서 "WorkFit Intranet" 및 "Enterprise Portal Suite"가 정상적으로 노출되는지 확인.
3. **퀵 도크 이관 확인**:
   - 우측 화면 가두리에 존재하던 주황색/초록색 세로 바가 완전히 사라졌는지 확인.
   - 상단 탑바 우측에 신규 아이콘 3종이 정상 렌더링되며, 클릭 시 해당하는 퀵 드로어 패널이 우측에서 부드럽게 슬라이드되는지 확인.
4. **닫기 UX 확인**: 퀵 도크 드로어의 헤더 부분 닫기 버튼이 `✕`로 수정되었는지 확인.
5. **테마 커스터마이저 작동 확인**:
   - 계정 프로필 클릭 ➔ "테마 설정(Custom)" 클릭.
   - 다양한 메인 테마 색상 및 포인트 색상(Teal, Blue 등)을 클릭 시, 화면 전체의 배경색 및 테마 포인트 색상(선택 테두리, 하이라이트 글자색 등)이 즉시 변형되는지 확인.
   - 브라우저를 새로고침한 이후에도 변경된 설정값이 `localStorage`를 통해 완벽히 유지되는지 확인.
6. **AX 위원회 제거 확인**: 부서 관리 및 조직도 등에서 'AX Committee' 관련 항목이 완벽히 정돈되었는지 대조 확인.
